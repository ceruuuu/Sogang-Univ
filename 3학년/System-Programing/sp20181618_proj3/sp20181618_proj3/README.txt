***실행 방법
컴파일은 "make"를 입력하여 컴파일해주십시오.
컴파일 이후 20181618.out이라는 실행 파일이 생성되며
"./20181618.out"을 입력하면 실행됩니다.

***파일 삭제
"make clean"을 입력하면 20181618.o, 20181618.out 두가지 파일이 삭제됩니다.

***프로그램이 종료되는 경우
opcode.txt. 파일이 현재 디렉토리에 존재하지 않을 경우 프로그램은 실행되지 않습니다.
